想法

- 使用布局的时候,该怎么添加东西
- 整体的布局中心是什么
- 随机数的生产(``nextInt()``函数)
- 怎样移动数字
- 怎样判断旁边是否有空位置
- 如果旁边有空位,怎样移动这个数到那个空位旁边
- 按键内部的函数的调用
